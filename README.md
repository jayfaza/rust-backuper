# The rust-backuper

This simple program allows you to make backups and save your data. You can just compile it and then you'll can to simply do backups on your pc.

# Usage:

| Flag        | Description                                                                                                                                             |
| ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- |
| -s or --src | Indicates which folder needs to be backuped.                                                                                                            |
| -d or --dst | Indicates in which folder to make a backup. '~/backup/' is the default folder.                                                                          |
| --nvim      | Instead of -d flag that is mandatory to be instructed you can indicate just '--nvim' and program will be using default path to neovim: '~/.config/nvim' |
